# trump-is-a-mess

Spring Boot/React App for Generating Trump Tweets using my markov-chain library. 


### Stack

* [ReactJS] - Front End
* [Sping Boot] - Backend
* [Java 8] - Backend

### Installation

Running this requires [Maven](https://maven.apache.org/install.html) v3.1+ to run because of the node js plugin.

Run maven to install and start the server.

```sh
$ cd trump-is-a-mess
$ mvn clean install
$ mvn spring-boot:run
```

then navigate to localhost:8080
